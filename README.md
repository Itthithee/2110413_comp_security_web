## Docker Compose

To start docker compose :

```bash
$ docker-compose up
```

For rebuild container:
```bash
$ docker-compose up --build
```

To stop docker compose :

```bash
$ docker-compose down
```

## PS1 
*** Don't panic if you see a bunch of warning when you first build container (most of it is about dependency) ***
